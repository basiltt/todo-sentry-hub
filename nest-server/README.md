
# NestJS Backend for Todo Application

This is the NestJS backend for the Todo application with role-based access control.

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Features

- Authentication (login/register)
- Todo management with RBAC
- JWT-based authorization
